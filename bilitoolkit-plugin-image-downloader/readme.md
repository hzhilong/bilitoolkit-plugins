# 图片下载

## 项目简介

[哔哩工具姬](https://github.com/hzhilong/bilitoolkit) 插件。

快速下载视频封面、专栏图片、用户头像、动态和评论表情包。

## 截图

![screenshot-01.png](doc/screenshots/screenshot-01.png)

## 安装使用

1. 安装并启动 [哔哩工具姬](https://github.com/hzhilong/bilitoolkit)；
2. 在插件市场中搜索并安装「图片下载」；
3. 打开「图片下载」即可使用该项目；
